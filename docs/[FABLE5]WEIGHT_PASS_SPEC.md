# [FABLE5] 加重·质感升级规格（毛坯房 → 考究案卷）

> 作者：Fable 5 · 2026-07-17 · 依据用户 weapp 真机截图（锦囊页，iOS）
> 病因：卡面与纸底对比几乎为零 → 全屏只剩 1px 线框；无填色、无阴影、无重量 → "毛坯房/简单线条"。
> 用户要"更重的视觉"。目标：保留直角案卷identity，但把每个元素做出分量与考究，去掉毛坯感。
> 不改 IA、不改功能；纯视觉加重。所有改动 **weapp 与 h5 都必须成立**（真机为准）。

---

## 1. 拉开层次：卡片必须"浮"出来（最高优先）

现状 --surf(#FAF7EF) vs --bg(#F0EBE0) 亮度差 ~2%，卡片消失。改：

- **加深纸底、提亮卡面，明确拉开对比**：
  - `--bg: #E9E2D3`（更暖更深的纸底，作"桌面/衬底"）
  - `--surf: #FBF9F3`（卡面近象牙白，明显亮于底）
  - `--surf-2: #F1ECE0`（次级面）
  - 深色模式同步：`--bg:#17150F` / `--surf:#221F17`（卡面明显亮于底）
- **卡片加纸张阴影 + 更实描边**（`.proto-card`）：
  ```
  border: 1px solid var(--hair-2);
  box-shadow: 0 1px 0 rgba(255,255,255,.6) inset,   /* 顶部微高光，纸的厚度 */
              0 2px 3px rgba(60,45,20,.05),
              0 14px 28px -16px rgba(60,45,20,.14);   /* 柔和落影 */
  ```
  weapp 支持 box-shadow；真机验证落影可见。深色模式落影用更深 rgba、去掉 inset 高光。
- 结果：每张卡读成一张有厚度的案卷纸，而非空心线框。

## 2. 本命色上重量：填色，不描边

现状 accent 只出现在细线圆圈和浅色小字。改为"朱批 + 印记"式的实体存在：

- **类别圆标（履/时/料/法/报/创 等所有圆形首字标）**：由"细线空心圆"改为
  **实心本命色圆**（`background: var(--ac)`；字 `var(--onac)` 纸白；字重 600），或
  软底变体（`background: var(--acg)` + `var(--ac)` 粗字 + 1px --ac 描边）。默认用实心，视觉最重。
- **题眉 kicker**（存你的家底/方案版本 等）：左侧加一枚 **实心本命色小方印**（6×6~8×8px 实块）
  或一段 2px 高 24px 宽的实底 accent 短横，压在题眉前，让分区有"落章"感。
- **章节分隔**：单薄 1px hairline 升级为"居中一枚 accent 小菱/短横 + 两侧渐隐细线"的考究分隔（至少大分区用）。

## 3. 按钮/CTA：实底，不空框

- 现"去问策"等是空心描边框（像占位）。所有 CTA 一律 **实底**：
  主行动 `.proto-btn` 已实底 accent；次级/空态 CTA 也改实底（ink 实底 `var(--tx)` + `var(--onac)` 字，
  或 accent 实底），保留 2px 直角 + 字距。**禁止**再出现空心线框按钮当主 CTA。
- 空态（待上传/待出策/待创作）：不要只留灰字，配一枚实底小 CTA + 一句军师语声引导，让空态也"完成"。

## 4. 案卷character：印章 + 边角（去毛坯的点睛）

- **印章**：报告卡/履历/命盘卡等关键产出，右下角盖一枚本命色实心方印（含单字，如 断/策/军/师），
  10~16px，轻微旋转 -4deg，作"钤印"。这是全站记忆点与"考究"来源。
- **案卷边角**：重点卡（军情主要矛盾卡、履历卡）四角加"L 形角线"（案卷装订角），2px accent 短角线，
  绝对定位四角。低成本、高character。
- **水印巨字**：opacity 由 .06 提到 .09~.12（真机可感），确保渲染且 pointer-events:none（勿拦点击）。

## 5. 纸的质感（可选但强烈建议）

- 纸底与卡面加**极轻纸纹**：一张 tileable 的低对比噪点/纤维纹 base64 作 background（叠在 --bg/--surf 上，
  opacity 极低）。weapp 用小尺寸 base64 png 平铺；控制体积。做出"纸"而非"纯色块"。
- 若 weapp 平铺纹理不稳，退而用 1~2 层极淡 radial/linear 叠加模拟纸的不均匀，也比纯平色有质感。

## 6. 字阶与密度微调

- 标题（30px）保持；正文行距略收（1.8→1.75）让内容更"结实"不松散。
- 卡内信息不足时（空态）适当抬高卡的最小高度并居中内容，避免"半空大白卡"的毛坯感。

---

## 验收（真机 weapp 为主）

对照本次截图，改后应达到：卡片明显"浮"在纸底上（有影有厚度）；圆标/按钮/题眉有本命色实体存在；
关键卡有印章与边角；水印可感；整屏读起来是"考究的案卷册"，不是"米色底上一堆细线框"。
weapp（真机）与 h5 双端一致；直角案卷identity 不变（仍是 2px 直角、宋体、墨+本命色）。

## 落地顺序
先地基（app.scss：令牌对比、.proto-card 阴影、.proto-btn 实底、印章/边角/题眉印记工具类、水印 opacity、纸纹），
再逐页把空心圆标→实心、空框 CTA→实底、关键卡加印章边角。weapp build 绿 + 真机复验。
